    CF4M

    Minecraft客户端框架

    快速上手 →

    简单

    不用写Event、Module和Setting等等.

    小

    小于40KB.

    轻量级

    仅用了Guava库(Minecraft自带.

    添加到build.gradle文件末尾

    allprojects {
	repositories {
		maven { url 'https://maven.enaium.cn' }
	}
}

    添加依赖项

    dependencies {
	implementation 'cn.enaium.cf4m:cf4m:最新版本'
}

    最新版本= (opens new window)

    # Fabric

    dependencies {
	modImplementation 'cn.enaium.cf4m:cf4m-fabric:至少版本'
}

    添加到fabric.mod.json

    "depends": {
    "cf4m": ">=至少版本"
}
